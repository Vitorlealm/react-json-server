# Cadastro de Tarefas (to-do list)

Aplicacao em React + Vite que consome uma API fake criada com JSON Server.

## Como instalar as dependencias

O projeto fica na pasta `react-json-server`. Entre nela e instale as dependencias:

```
cd react-json-server
npm install
```

## Como rodar a aplicação

Ainda dentro da pasta `react-json-server`, rode:

```
npm run dev
```

Esse comando sobe os dois servicos ao mesmo tempo:

- API (JSON Server): http://localhost:3000
- Aplicacao (Vite): http://localhost:5173

Abra http://localhost:5173 no navegador para usar a aplicacao.
